## Sobre o Projeto

**Título Completo:** Mulsemídia encontra a Realidade Virtual Imersiva: Uma Abordagem para Criação de Experiências Multissensoriais Imersivas, Interativas e Acessíveis.

A Realidade Virtual Imersiva (iVR) permite criar ambientes complexos e realistas. No entanto, muitas experiências focam apenas em visão e audição. Este projeto busca ampliar a gama de estímulos sensoriais (mulsemídia) para aumentar a imersão e promover a acessibilidade, permitindo que estímulos alternativos sejam escolhidos conforme o perfil do usuário (ex: PcD).

Para superar a complexidade de desenvolvimento desses ambientes, propomos uma abordagem declarativa baseada na extensão da linguagem NCL e do motor de apresentação Guaraná (originalmente focado na TV 3.0).

<br>

## Objetivos

O objetivo geral é oferecer abstrações para a autoria de aplicações iVR multissensoriais, permitindo que autores com conhecimento básico em programação criem experiências ricas.

Os objetivos específicos incluem:

1.  **Consolidação de Parceria:** Fortalecer a colaboração entre CEFET-RJ, UFC e IFCE, evoluindo artefatos de pesquisas anteriores sobre orientação e mobilidade.
2.  **Extensão da Linguagem NCL360:** Novos elementos na linguagem NCL para representar características de experiências multissensoriais em RV.
3.  **Motor de Apresentação Mulsemídia:** Criação de novos módulos no motor Guaraná para suporte aos conceitos de mulsemídia em iVR.
4.  **Integração com EdKit:** Incorporar funcionalidades de reconhecimento de gestos, feedback multimodal e proveniência de dados do framework EdKit (Unity).
5.  **Assets para Unity:** Desenvolvimento de assets que permitam definir comportamentos declarativos dentro da engine Unity.
6.  **Casos de Uso:** Criação de duas experiências interativas, sendo ao menos uma voltada para pessoas com deficiência visual (utilizando som, vibração, cheiro, vento, etc.).
